Implementation notes

Progress ring (A1 / A2) — SVG ring using stroke-dashoffset on a circle with r=21 (circumference ≈ 131.95px). The same getLifeBarColor() logic drives the stroke color. A CSS @keyframes animation draws the ring from 0 → target on card entry, slotting into the existing IntersectionObserver callback. A1 replaces the horizontal bar entirely; A2 keeps the bar hidden and overlays the ring on the card image.

Habitat & diet badges (B) — Requires two new fields per species in data.js: habitatTypes: ["ocean", "coastal"] (array, picks from a fixed set) and dietType: "apex-predator" (single value). Badge color and icon are keyed to each type string. Rendered as .badge-row pills between the sci name and threat badge.

Combined (C) — Ring replaces the bar; badges sit below the sci name. Total card height ~30px taller than current. The card body stays a single-column flex column — no layout restructuring needed.
